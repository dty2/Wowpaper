## 简介

WowPaper 是一个运行在 Wayland 环境下的壁纸播放器

它支持图片作为静态壁纸, 视频作为动态壁纸

Use the Wowpaper, Be a Wow Maker!

> [!IMPORTANT]
> WowPaper采用使用wlr-protocols拓展协议实现窗口置底
> 因此只能在支持该协议的合成器上(如hyprland)工作
> 无法在不支持"wlr-protocols"的合成器或桌面环境(如GNOME)上工作

更多详细信息请看[官网](https://wow-paper.fun)
